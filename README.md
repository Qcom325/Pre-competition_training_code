# Pre-competition_training_code
寒假线上集训手册2026 v1.0：
1：队长每周定期发布1、2个任务包，春节期间可以停训，总数为6+周（队长可增加6周后的训练），队员完成任务，队长检查，获得积分；
2：积分太低的同学（无故缺训多次），比赛前可由替补队员替换
3：队员按以下约定提交任务交付物：（代码+文档，不仅是题目ac）
  1）代码提交：所有代码必须提交至团队Git仓库的对应周次目录下（如 `Week1/PackageA-1/`）。文件可以“题目编号-姓名缩写”格式命名。
  2）学习笔记：每个任务包要求的学习笔记/总结，必须在指定的共享文档中，于本人名下填写。笔记需包含解题思路、遇到的核心难点、时间与空间复杂度分析、学到的技巧或坑点。
  3）完成标准：“完成”指所有必做题在OJ上正确通过（Accepted）、学习笔记有内容。
  4）优秀代码标准：结构清晰、命名规范、关键步骤有注释、算法效率优、有独特见解。
